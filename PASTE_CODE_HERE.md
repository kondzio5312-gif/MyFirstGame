# 📋 INSTRUKCJA COPY-PASTE KODU

## Procedura:

1. **Otwórz plik w Roblox Studio**
2. **Zaznacz cały kod** (Ctrl+A)
3. **Skopiuj** (Ctrl+C)
4. **Wklej poniżej między backticks oraz zastąp `PASTE_HERE`**
5. **Powiedz którym plikiem jesteś** (np. "Config gotowy!")

---

## Format:

```
KTÓRA TO ŚCIEŻKA? (np. shared/Config.luau)
```

```lua
-- PASTE_HERE
```
